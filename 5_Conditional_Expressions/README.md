## If - Else

### Comparison Check

**-gt** : **G**reat
**-lt** : **L**ittle
**-ge** : **G**reat **E**qual
**-le** : **L**ittle **E**qual
**-eq** : **Eq**ual
**-ne** : **N**ot **E**qual

### Array Check

**-z** : Null Array 
**-n** : Defined Array
**=**  : Equal Array
**!=** : Not Equal Array

### File Check

**-f** : File is
**-s** : File is not null
**-r** : File is Readable
**-w** : File is Writeable
**-x** : File is runnable
**-h** : SoftLink
**-c** : Character Device
**-b** : Blok Device

### Logical Check

**-a** : And
**-o** : Or
**!**  : Not
